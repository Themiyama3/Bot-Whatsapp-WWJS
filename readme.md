# Bot Whatsapp WWJS

Whatsapp bot using Whatsapp-Web.js library


## Installation


```bash
  # mkdir mybot
  # cd mybot
  # git clone https://github.com/Zynfinity/Bot-Whatsapp-WWJS
  # npm install
  # node index.js
```
